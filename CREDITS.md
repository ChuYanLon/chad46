# Credits

chad46 stands on the shoulders of these projects:

## Upstream

- **[NvChad/base46](https://github.com/NvChad/base46)** — All 94 themes, 44 plugin highlight integrations, and the base_30/base_16 color system. Synced daily.
- **[NvChad/ui](https://github.com/NvChad/ui)** — Type definitions and UI components.

## Inspiration

- **[tokyonight.nvim](https://github.com/folke/tokyonight.nvim)** — Auto-detect integration system.

## Plugin Integrations

Highlights are generated for these plugins. See README for the full list.

- **Lualine** / **Heirline** / **Bufferline** — Statusline and tabline adapters (via NvChad)
- **vim-airline** / **lightline.vim** — Statusline themes (94 pre-generated)
- **coc.nvim** — LSP completion highlights
- **NERDTree** — File explorer highlights

## Colorscheme Generation

- Static `.vim` files generated by `nvim --headless`, providing Vim compatibility
- cterm color approximation via xterm 256-color cube mapping
- Manifest-based incremental generation to reduce unnecessary writes
